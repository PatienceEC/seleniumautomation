About Me

I'm a QA Automation Engineer with hands-on experience building end-to-end test frameworks using Playwright and Selenium. I work across the full testing lifecycle — from designing test strategies and writing automation suites to integrating them into CI/CD pipelines.

My focus is on writing tests that are maintainable, fast, and actually catch bugs before users do. I follow the Page Object Model pattern and prefer automation that lives alongside the code it tests.

I'm currently open to QA Automation Engineer and SDET roles where I can help teams ship with confidence.
